# Quay Demo Repository

This is a demo repository for **Quay**, a macOS menu bar app that syncs local folders with GitHub in one click.

Quay is designed for writers, bloggers, and anyone who works with static site generators like Astro, Hugo, or Jekyll. Instead of learning Git commands, you just click Sync.

## What's in this repo

- `blog/` — A sample blog with a few posts
- `notes/` — Some quick notes

Feel free to edit any file and use Quay to sync your changes back to GitHub.
